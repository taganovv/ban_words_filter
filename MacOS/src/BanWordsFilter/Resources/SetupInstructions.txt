Настройка приложения BanWordsFilter

Фильтр работает от вашего аккаунта стримера. Отдельный бот-аккаунт не нужен. Вся информация хранится локально, ваши данные в безопасности!

────────────────────────────────────────
1. ПОДГОТОВКА
────────────────────────────────────────

• Используйте аккаунт, на котором вы стримите
• Включите двухфакторную аутентификацию (2FA):
  https://www.twitch.tv/settings/security
  Без 2FA сайт dev.twitch.tv может не открыться

────────────────────────────────────────
2. СОЗДАТЬ ПРИЛОЖЕНИЕ НА TWITCH
────────────────────────────────────────

1. Откройте https://dev.twitch.tv/console
   Войдите аккаунтом стримера

2. Нажмите «Создать расширение»

Укажите любое имя вашего расширения. Затем нажимайте кнопку "Продолжить"

3. У вас открывается окно с версиями вашего расширения, ничего делать в нем не нужно! Переходите по кнопке "Настройки расширения"

4. !!! OAuth Redirect URL - данное поле по стандарту имеет значение типа "https" исправьте на "http" и нажмите кнопку сохранить изменения !!!

5. Далее создайте секретный код клиента Twitch API

6. Вставьте Client ID и Client Secret в поля приложения
Client ID - это Идентификатор клиента в правом верхнем углу
Client Secret - это Секретный код клиента Twitch API

────────────────────────────────────────
3. ПОЛУЧИТЬ OAUTH TOKEN
────────────────────────────────────────

1. Вставьте Client ID в поле приложения
2. Нажмите «Получить OAuth Token»
3. В браузере нажмите Authorize (аккаунтом стримера)
4. После авторизации адресная строка будет такой:

   http://localhost/#access_token=ДЛИННАЯ_СТРОКА&scope=...

   Страница может не загрузиться — это нормально.

5. Скопируйте текст между access_token= и &
   Или вставьте всю строку из адресной строки —
   приложение само извлечёт токен.

6. Вставьте в поле OAuth Token

────────────────────────────────────────
4. ПРОВЕРКА И ЗАПУСК
────────────────────────────────────────

1. Нажмите «Проверить токен» → должно быть «Токен OK»
   Username, User ID и Channel заполнятся автоматически

2. Channel = ваш логин Twitch (канал, где идёт стрим)

3. Нажмите «Сохранить»

4. Нажмите «▶ Старт»

5. В чате вашего канала появится:
   Ban Words Filter Activated!

6. Проверка: напишите в чат !blockstatus
   Ответ Filter ON получите только вы (broadcaster)

────────────────────────────────────────
5. ЕСЛИ НЕ РАБОТАЕТ
────────────────────────────────────────

• Нет сообщения Ban Words Filter Activated!
  → вкладка «Лог», проверьте ошибки
  → нажмите «Проверить токен» ещё раз

• Redirect URI mismatch
  → в dev.twitch.tv укажите http://localhost
  → в приложении выберите тот же Redirect

• Фильтр не банит
  → Channel и Username = ваш логин
  → токен получен аккаунтом стримера

• Приложение закрыли → фильтр выключился
  → запускайте перед каждым стримом
